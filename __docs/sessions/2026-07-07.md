## Data Model

`docs/DATA_MODEL.md` updated with finalized decisions:

- **Season** added as a first-class entity (`id, org_id, name, is_active`). Teams and Signups reference `season_id`.
- **SignupEligibleRole** join table replaces the former `eligible_roles` array field on Signup.
- **SlotResponse** unique constraint: `(user_id, slot_id)`. DIRECT_CLAIM allows multiple slots per signup; RANKED_CHOICE adds a rank per slot.
- **Audit fields** (`created_at`, `updated_at`, `created_by`, `updated_by`, `deleted_at`) on all non-pivot entities. Pivot tables excluded.
- `SlotResponse.declined` is admin-only (RANKED_CHOICE rejection). Volunteers withdraw by deletion while status is `open`.
